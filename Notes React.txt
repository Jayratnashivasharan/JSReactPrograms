State in React:
State is a container to store data like variables
This is mutable and dynamic.
We have to import it when if we want to use it.
It re-render component automatically  so that data can visible on UI.

State in React:
import {useState} from "react";
function App(){
	const [Fruits , setFruits]= useState{"Apple"};
	const handleFruits=()=>{
	setFruits="Banana";
}
return (
<div>
	<h1>{Fruits}</h1>
	<button onclick={handleFruits}>Change Fruit Name </button>
</div>
);
}
